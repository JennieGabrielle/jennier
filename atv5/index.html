Resumo — Flexbox e Grid CSS

FLEXBOX (Flexible Box Layout)

Ideal para layouts unidimensionais (linha ou coluna)

Principais funcionalidades (propriedades do container)

  • display: flex — ativa o flexbox.

flex-direction

  • row (padrão) – itens em linha
  • row-reverse
  • column – itens em coluna
  • column-reverse

flex-wrap

  • nowrap (padrão)
  • wrap
  • wrap-reverse

flex-flow

Atalho para flex-direction + flex-wrap

justify-content

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

align-items

  • stretch (padrão)
  • flex-start
  • flex-end
  • center
  • baseline

align-content

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • stretch

Propriedades dos itens (filhos)

  • flex-grow — define quanto o item cresce
  • flex-shrink — define quanto o item diminui
  • flex-basis — tamanho inicial do item
  • flex — atalho grow + shrink + basis
  • align-self — alinha um item individualmente
  • order — muda a ordem dos itens

Vantagens do Flexbox

  • Melhor para alinhamento preciso de elementos.
  • Ótimo para menus, cards, componentes, botões, headers e footers.
  • Perfeito para elementos que precisam ajustar ao tamanho do conteúdo.
  • Facilidade para centralizar elementos.

CSS GRID

Ideal para layouts bidimensionais (linhas e colunas)

Principais funcionalidades do container

  • display: grid — ativa o grid.
  • grid-template-columns — define colunas.
  • grid-template-rows — define linhas.
  • grid-template-areas — cria áreas nomeadas.
  • grid-auto-rows / grid-auto-columns — tamanhos automáticos.
  • grid-auto-flow — fluxo dos itens: row, column, dense.
  • gap / row-gap / column-gap — espaçamento.
  • justify-items — alinha itens horizontalmente.
  • align-items — alinha itens verticalmente.
  • place-items — atalho para justify-items + align-items.
  • justify-content — alinha o grid como um todo horizontalmente.
  • align-content — alinha o grid verticalmente.
  • place-content — atalho justify-content + align-content.

Propriedades dos itens (filhos)

  • grid-column / grid-row — define onde o item fica.
  • grid-area — usa áreas nomeadas.
  • justify-self — alinha item horizontalmente.
  • align-self — alinha item verticalmente.
  • place-self — atalho justify-self + align-self.

Funções importantes do Grid

  • repeat() — repetição automática.
    Ex.: repeat(3, 1fr)
  • minmax() — intervalo de tamanho.
    Ex.: minmax(200px, 1fr)
  • auto-fit / auto-fill — responsividade automática.
    Ex.: repeat(auto-fit, minmax(250px, 1fr))

Vantagens do CSS Grid

  • Melhor escolha para layouts completos de página.
  • Organiza em linha e coluna ao mesmo tempo.
  • Permite criar áreas como header, main, aside e footer.
  • Responsividade automática com auto-fit e minmax.
  • Ideal para galerias, dashboards e layouts complexos.
